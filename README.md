# Eda
EDA of 500 Person Gender-Height-Weight-Body Mass Index

About Dataset

Gender : Male / Female

Height : Number (cm)

Weight : Number (Kg)

Index :

0 - Extremely Weak

1 - Weak

2 - Normal

3 - Overweight

4 - Obesity

5 - Extreme Obesity

dataset: https://www.kaggle.com/datasets/yersever/500-person-gender-height-weight-bodymassindex
